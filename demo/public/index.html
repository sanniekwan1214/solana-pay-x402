Solana Pay x402 mainnet

Hybrid HTTP 402 Payment Demo — Two flows, one middleware

x402 v2 (Automatic)

  • Browser wallet signs tx
  • Facilitator submits tx
  • Facilitator pays gas
  • Fully programmatic
Best for: dApps, APIs, automation

Solana Pay (QR Code)

  • Mobile wallet scans QR
  • User submits tx on-chain
  • User pays gas
  • Works on any device
Best for: mobile, retail, POS
